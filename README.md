# Anonymous Confessions

Anonymous Confessions is a frontend web application that enables users to share thoughts and experiences anonymously in a structured and interactive feed-based interface. The platform is designed with a focus on clean architecture, scalability, and modern React development practices.

Built using React.js and Tailwind CSS, the application implements a component-driven architecture with centralized state management via Context API. It supports content creation, category-based filtering, engagement through likes, and persistent client-side storage using localStorage to maintain data across sessions.

The interface is fully responsive and optimized for usability across devices. Utility functions are used to maintain clean and reusable logic, while real-time relative timestamps enhance user experience by providing contextual post timing.

This project demonstrates practical implementation of frontend engineering concepts including state management, reusable component design, data persistence strategies, and UI/UX considerations in modern web applications.

## Core Features

* Anonymous content creation and feed display
* Category-based filtering system
* Like interaction with persistence
* Real-time relative timestamps
* Responsive UI with Tailwind CSS
* Global state management using Context API
* LocalStorage-based data persistence

## Tech Stack

• React.js 
• Tailwind CSS 
• JavaScript (ES6+) 
• Context API 
• LocalStorage

## Architecture

The project follows a modular component-based structure with separation of concerns between UI components, state management, utilities, and data handling. It is designed to be backend-ready for future integration with REST APIs or real-time services.
